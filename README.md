## How to run
* Make sure you have __PyQt5__ and __fbs__ installed

* If they are not installed on your computer:
    * Install __PyQt5__

        ```
        > sudo pip3 install pyqt5
        ```

    * Install __fbs__

        ```
        > sudo pip3 install fbs
        ```

* Run with this command
    ```
    > fbs run
    ```
